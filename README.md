# 사장님의 사장님 — 폐업 동행 코치 (P0 프로토타입)

> **"전 망한 게 아니에요, 졸업한 거예요."**
> 폐업한 사장님이 폐업할 사장님을 4주간 1:1 동행하는 한국 최초 "잘 끝내는 법" 양면 시장 플랫폼.

## 화면 구성 (3 SPA)

1. **`/diagnose`** — 폐업 진단 30초: 업종·연차·직원수·임대상황 입력 → AI가 30개 절차 중 본인에게 해당하는 18~22개를 골라 4주 캘린더 자동 생성 (Claude Haiku 4.5)
2. **`/match`** — 코치 매칭: 업종·지역·실패유형 3축 필터, **박용군(족발 10년·평택) 1호 코치 카드** + 가상 코치 5명
3. **`/resume`** — 실패 경력서 자동 생성: 폐업 사유 입력 → AI가 "도전 경력서" 양식으로 출력 (재도전 펀드 신청 첨부용)

## 스택

- Next.js 14.2 (App Router) + React 18
- @anthropic-ai/sdk (Claude Haiku 4.5 직접 호출)
- Vanilla CSS (3시간 사양, 빠른 프로토타입)

## 로컬 실행

```bash
npm install
cp .env.example .env.local
# .env.local 파일에 ANTHROPIC_API_KEY 입력
npm run dev
# http://localhost:3000
```
